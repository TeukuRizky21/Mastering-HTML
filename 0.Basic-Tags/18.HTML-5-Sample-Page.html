Saat kita mau bikin sebuah aplikasi dengan memakai library React, kita akan ketemu dengan sebuah “props” singkatan dari properties. Pada react, props ini digunakan untuk komunikasi antara Parent component dengan Child component. Sebagai contoh , kita memiliki 3 file dalam satu folder kita beri nama tiap filenya index.js, Buah.js, Tanaman.js, berikut kodingannya :


pada file index.js


Pada file Tanaman.js


Pada file Buah.js


Dari kodingan diatas di layar akan muncul hasil : Buah apa yang kamu suka? Saya suka buah Apel Pada contoh diatas ada 2 component yang disediakan, yaitu component Tanaman, dan Buah. Tanaman sebagai parent, dan Buah sebagai childnya. Pertama, setelah memanggil component child “Buah” Kita bikin sebuah attribute didalamnya bernama “namaBuah” dan kemudian kita kasih nilai “Apel” . Kemudian kita panggil attribute “namaBuah” di child component dengan menggunakana props, {props.namaBuah}, selesai, dan muncul hasilnya. Perlu diingat, jika kita memakai functional component, jangan lupa untuk memanggil props sebagai parameter sebelum arrow function. Jika ingin tau lebih jauh tentang props, bisa baca di dokumentasi resminya disini
